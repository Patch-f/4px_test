## install

```git clone https://github.com/Patch-f/4px_test```

#### Install dependencies

```composer install```
```composer update```

#### Make link

```php artisan storage:link```

#### Make migrations

```php artisan migrate```

#### If you need test data

```php artisan db:seed```
